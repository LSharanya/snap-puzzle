Turn any screenshot into a jigsaw puzzle. Drag the pieces back together.

What it does
Take a photo, upload an image, or paste a screenshot. The app chops it into puzzle pieces. You drag them from the tray onto the board until they snap back together.

How to run
Open the folder in VS Code

Install Live Server extension

Right-click index.html → Open with Live Server

That's it. Camera works this way.

Files
text
index.html   # The page
style.css    # Colors and layout
script.js    # App logic
puzzle.js    # Jigsaw math
pieces.js    # Drag and snap
How to play
Pick an image (camera, upload, or paste)

Pick difficulty (3×3 to 6×6)

Drag pieces from bottom tray to the board

Pieces snap when close to home

Timer and moves count

Hint button shows ghost image
